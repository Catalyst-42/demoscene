Demoscene information
Демосцена (или же просто DS) - сайт, где каждый может оставлять комментарии без идентификации пользователя. 

Про форматирование
-//-

Стандартные:
Stroke     <- [s] Stroke [/s]
Underlined <- [u] Underlined [/u]
Bold       <- [b] Bold [/b]
Italic     <- [i] Italic [/i]

Кастомные:
rainbow    <- [rainbow] rainbow [/]
magic      <- [magic] magic [/]

golden     <- [gold] golden [/]
bronze     <- [bronze] bronze [/]
silver     <- [silver] silver [/]

jumping    <- [jump] jumping [/]
shaking    <- [shake] shaking [/]

Цвета:
magenta    <- [#f0f] magenta [/]
lime       <- [#ABE15B] lime [/]
