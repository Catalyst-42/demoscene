Demoscene information
Демосцена - сайт, где каждый может анонимно оставлять комментарии.  
Это обновлённая версия сайта, итоги первого сезона можно посмотреть тут.
Код сайта открыт для всех желающих.

Спасибо, что соблюдаете правила приличия, общаетесь вежливо и не затрагиваете острые темы. Все комментарии, содержащие брань, или ломающие работу сайта будут отредактированы или удалены. 

Про форматирование
Вы можете вставлять в комментарии модификаторы текста.
Модификаторы, которые не конфликтуют с анимациями, можно комбинировать. (Модификаторы из разных групп)

Обратите внимание, что все пробелы учитываются! Так что правильное использование тегов в контексте будет примерно вот таким:

Ничего [u]себе[/u], какой [#8CE10B][jump]странный[/][/] [#555]сайт[/]...  

Теги
Стандартные:
Stroke     <- [s] Stroke [/s]
Underlined <- [u] Underlined [/u]
Bold       <- [b] Bold [/b]
Italic     <- [i] Italic [/i]

Кастомные:
rainbow    <- [rainbow] rainbow [/]
magic      <- [magic] magic [/]

golden     <- [gold] golden [/]
bronze     <- [bronze] bronze [/]
silver     <- [silver] silver [/]

blinking   <- [blink] blinking [/]
jumping    <- [jump] jumping [/]
shaking    <- [shake] shaking [/]

Цвета:
magenta    <- [#f0f] magenta [/]
lime       <- [#8CE10B] lime [/]

         <- [img] ./images/DemosceneLogo.png [/img]
